## Bugs resolvidos hoje (Sessão 9.4) — JÁ COMMITADOS

Commit `89e134a feat(auth+infra): destrava chain de bugs AAD + SqlClient`:
1. AAD claim `app_tenant_id` tem **3 formas** (curta P1, longa ID token, `extn.` access token v2)
2. SqlClient v6+ separou auth providers → token explicit via Azure.Identity
3. Alpine .NET sem ICU → `apk add icu-libs` + `DOTNET_SYSTEM_GLOBALIZATION_INVARIANT=false`
4. MSAL `loginPopup` engolia erros → banner DOM selecionável

## Schemas/contratos compartilhados

### Bicep params (resultado do Wave 1.A — input pro Wave 2.D)

```bicep
@description('Python runtime version (default 3.13 — wheels estaveis).')
param pythonVersion string = '3.13'

@description('CORS origins adicionais ao backend FQDN (dev local).')
param additionalCorsOrigins array = []

@description('Pula prepdocs.py postdeploy.')
param skipPrepdocs bool = false

@description('Habilita chat na UI (Lab Intermediario ativa).')
param enableChat bool = false

@description('Audience format aceito tickets-service.')
@allowed(['v2', 'v1', 'both'])
param tokenAudienceFormat string = 'v2'

@description('SQL autoPauseDelay minutos (-1 = sempre online).')
param sqlAutoPauseDelay int = -1
```
